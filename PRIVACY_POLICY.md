# 隐私政策 / Privacy Policy

**最后更新日期 / Last Updated: 2026-01-18**

## 中文版

### 概述

"AI上网助手" 是一款浏览器扩展，旨在帮助用户翻译网页内容。我们非常重视用户隐私，本隐私政策说明了我们如何收集、使用和保护您的信息。

### 信息收集

#### 我们收集的信息

1. **用户设置**：您在扩展中配置的偏好设置（如目标语言、API选择等），这些数据仅存储在您的本地浏览器中。

2. **翻译内容**：当您使用翻译功能时，选中的文本会被发送到您选择的翻译服务进行处理。

#### 我们不收集的信息

- 我们不收集任何个人身份信息
- 我们不追踪您的浏览历史
- 我们不存储您翻译过的内容
- 我们不使用任何分析或追踪工具

### 数据使用

- **本地存储**：所有用户设置使用 Chrome Storage API 存储在您的本地设备上。
- **翻译请求**：根据您的配置，翻译请求可能发送到以下服务之一：
  - Ollama（本地运行，数据不离开您的设备）
  - OpenAI API（受 OpenAI 隐私政策约束）
  - 其他您配置的 API 服务

### 权限说明

本扩展请求以下权限：

| 权限 | 用途 |
|------|------|
| `activeTab` | 访问当前标签页以提供翻译功能 |
| `storage` | 保存您的偏好设置 |
| `contextMenus` | 提供右键菜单翻译选项 |
| `scripting` | 在网页中注入翻译功能 |
| `host_permissions` | 访问网页内容以进行翻译 |

### 第三方服务

当您使用在线翻译 API 时，您的数据将受到相应服务提供商的隐私政策约束。我们建议您查阅：
- [OpenAI 隐私政策](https://openai.com/privacy/)

### 数据安全

- 所有设置数据仅存储在您的本地浏览器中
- 使用 Ollama 时，所有数据处理都在本地完成
- 我们不运营任何服务器来存储用户数据

### 儿童隐私

本扩展不针对 13 岁以下的儿童，我们不会故意收集儿童的个人信息。

### 政策变更

如果我们对隐私政策进行重大更改，我们会通过扩展更新通知用户。

### 联系我们

如果您对本隐私政策有任何疑问，请通过以下方式联系我们：
- GitHub Issues: [https://github.com/citysay/localla]
- 电子邮件: [youngsun@qq.com]

## English Version

### Overview

"AI Web Assistant" is a browser extension designed to help users translate web content. We take user privacy seriously, and this privacy policy explains how we collect, use, and protect your information.

### Information Collection

#### Information We Collect

1. **User Settings**: Preferences you configure in the extension (such as target language, API selection, etc.), stored only in your local browser.

2. **Translation Content**: When you use the translation feature, selected text is sent to your chosen translation service for processing.

#### Information We Do NOT Collect

- We do not collect any personally identifiable information
- We do not track your browsing history
- We do not store your translated content
- We do not use any analytics or tracking tools

### Data Usage

- **Local Storage**: All user settings are stored locally on your device using Chrome Storage API.
- **Translation Requests**: Depending on your configuration, translation requests may be sent to one of the following services:
  - Ollama (runs locally, data never leaves your device)
  - OpenAI API (subject to OpenAI's privacy policy)
  - Other API services you configure

### Permissions Explanation

This extension requests the following permissions:

| Permission | Purpose |
|------------|---------|
| `activeTab` | Access current tab to provide translation |
| `storage` | Save your preferences |
| `contextMenus` | Provide right-click menu translation option |
| `scripting` | Inject translation functionality into web pages |
| `host_permissions` | Access web page content for translation |

### Third-Party Services

When using online translation APIs, your data will be subject to the respective service provider's privacy policy. We recommend reviewing:
- [OpenAI Privacy Policy](https://openai.com/privacy/)

### Data Security

- All settings data is stored only in your local browser
- When using Ollama, all data processing is done locally
- We do not operate any servers to store user data

### Children's Privacy

This extension is not directed at children under 13, and we do not knowingly collect personal information from children.

### Policy Changes

If we make significant changes to this privacy policy, we will notify users through extension updates.

### Contact Us

If you have any questions about this privacy policy, please contact us:
- GitHub Issues: [https://github.com/citysay/localla]
- Email: [youngsun@qq.com]
